# HydroAlert — IoT-Based Flood Early Warning System

HydroAlert is an Internet of Things (IoT)–based flood early warning system developed as part of the Innovillage program.  
The system monitors water level data in real time using IoT sensors and presents the information through a web-based dashboard to support early flood mitigation.

---

## Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Chart library (e.g., Recharts / Chart.js)

**Backend**
- Node.js
- Express.js
- REST API

**Database**
- MongoDB (MongoDB Atlas)

---

## Getting Started
**Prerequisites**
- Node.js (v18 or later recommended)
- npm
- MongoDB (local or MongoDB Atlas)

---

## Backend Setup
```
cd hydroalert-backend
npm install
```
### Create a .env file based on .env.example
```
PORT=port_for_your_backend_server
```
### Run the backend server
```
npm run dev
```
